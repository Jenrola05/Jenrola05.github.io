Introduction to HTML

HTML, which stands for Hypertext Markup Language, is the standard markup language used to create web pages. It provides a structured way to organize content on the web by using a set of predefined tags.

Basic Structure of HTML Document

        
            <!DOCTYPE html>
            <html lang="en">

            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Introduction to HTML</title>
            </head>

            <body>
                <h1>Introduction to HTML</h1>
                <p>
                    HTML, which stands for Hypertext Markup Language, is the standard markup language used to create web pages.
                    It provides a structured way to organize content on the web by using a set of predefined tags.
                </p>
            </body>

            </html>

How Web Content is Structured and Presented

Text Content: Text content is placed within appropriate tags like p> for paragraphs, h1> to h6> for headings, and others. The structure and hierarchy of headings contribute to the organization and readability of the content. Images and Media: Images are embedded using the img> tag, and other media elements like audio and video have dedicated tags (audio>, video>). Attributes such as src are used to specify the source (URL) of the media. Links: Hyperlinks are created with the a> tag and the href attribute, defining the URL the link points to. Lists: Lists are created using ul> (unordered list) and ol> (ordered list) with li> (list item) elements. Structural Elements: Structural elements like div> and span> are used to group content, allowing for styling and layout control.
